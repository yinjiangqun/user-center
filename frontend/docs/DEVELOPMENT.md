## 技术栈要求

- Node.js版本: v18.0.0或更高
- Vue.js版本: Vue 3.x
- TypeScript版本: 5.0或更高
- 包管理器: npm 8.x或更高/yarn 1.22.x或更高

## 开发工具推荐配置

- **IDE**
  - Visual Studio Code (推荐)
  - WebStorm
- **VSCode推荐插件**
  - Vue Language Features (Volar)
  - TypeScript Vue Plugin (Volar)
  - ESLint
  - Prettier - Code formatter
  - GitLens

## 浏览器要求

- Chrome 最新版(推荐)
- Firefox 最新版
- Safari 最新版
- Edge 最新版
